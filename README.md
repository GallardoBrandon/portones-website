# Sitio: Instalación de Portones y Mercancía

Pequeña página de ejemplo para mostrar instalaciones de portones y la mercancía que vendes.

Archivos:
- index.html — página principal
- styles.css — estilos
- script.js — manejo del formulario

Cómo usar:
1. Abrir `index.html` en el navegador (doble clic o usar el comando en PowerShell abajo).

Comando (PowerShell):

```powershell
Start-Process index.html
```

Personalización rápida:
- Reemplaza las imágenes con tus fotos en la carpeta y cambia las rutas en `index.html`.
- Actualiza textos, precios y datos de contacto.

Siguientes pasos sugeridos:
- Añadir un formulario real que envíe emails (usando un servicio como Formspree o un backend).
- Subir a GitHub y activar GitHub Pages para publicar.
